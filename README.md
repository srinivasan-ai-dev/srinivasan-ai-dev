<summary><b>⚙️ &nbsp; SETUP GUIDE — Snake Contribution Graph (for repo cloners)</b></summary>

<br/>

To enable the animated snake contribution graph, add this file at `.github/workflows/snake.yml`:

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:

jobs:
  generate:
    runs-on: ubuntu-latest
    timeout-minutes: 10
    steps:
      - uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

Then reference it in the README as:
```
![Snake](https://raw.githubusercontent.com/srinivasan-ai-dev/srinivasan-ai-dev/output/github-contribution-grid-snake-dark.svg)
```
